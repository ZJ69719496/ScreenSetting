# ScreenSetting
大屏幕信号分发工具  用来控制多个屏幕组装的大屏幕  可以单独设置每个屏幕的信号和显示的内容，可以在同一个图形操作界面上即可实现对各种信号窗口的切换、调用、控制、应用和管理。实现叠加漫游开窗的显示应用
